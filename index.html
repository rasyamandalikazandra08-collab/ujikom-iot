Tentang Sistem Monitoring

Platform monitoring ini dibangun untuk membantu Anda mengawasi perangkat IoT dengan mudah. Dengan integrasi Firebase, semua data tersinkronisasi secara realtime tanpa perlu konfigurasi rumit. Kami menyediakan ekosistem lengkap mulai dari embedded hingga dashboard modern.

  • Mudah diintegrasikan dengan ESP32 / ESP8266
  • Mendukung banyak sensor secara bersamaan (DHT, ultrasonic, dll)
  • Dashboard yang dapat dikustomisasi dengan tema gelap/terang
  • Notifikasi push & alert via email (segara hadir)
